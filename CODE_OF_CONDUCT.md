# Code of Conduct

## Our Pledge
We pledge to create a welcoming and respectful environment for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Being respectful
- Accepting constructive criticism
- Showing empathy

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the standards.

## Scope
This Code of Conduct applies to all project spaces.

## Enforcement
Instances of abusive behavior may be reported to project maintainers.

## Enforcement Guidelines
- Warning
- Temporary ban
- Permanent ban (if necessary)

This Code of Conduct is adapted from Contributor Covenant.
